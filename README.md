# Personal Portfolio Website

A modern personal portfolio website showcasing my projects, skills, research work, and professional experience.

## 🚀 Features

- Responsive modern UI
- Projects showcase
- Skills and technology stack
- Research and publications section
- Contact information
- Resume download / view

## 🛠 Tech Stack

- React
- JavaScript
- Tailwind CSS
- Framer Motion

## 📂 Sections

- Home
- About
- Skills
- Projects
- Research
- Certifications
- Contact

## ⚙ Installation

Clone the repository

git clone https://github.com/yourusername/portfolio.git

Go to project folder

cd portfolio

Install dependencies

npm install

Run development server

npm run dev
